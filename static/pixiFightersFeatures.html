Pre Loading

Pre Loading assets makes it easy on the browser to load them later
You can also rename as you load them for quality of life improvement
It makes a big difference on larger project but is still noticeable on smaller ones

Parallax effect

A parallax effect is a superposition of different layers to create an animated background
Each layer moves at a different speed to create a depth illusion
It makes for a much more immersive experience as a player

The app ticker makes it running easily while the pre-loader makes it easy on the performance side

Containers

Containers makes it easy to create structures that you can later re-use or re-arrange easily
It works as parent-child hierarchy and you can easily mask or filter a child from a container
